# Interactive Drum Pad

A browser-based interactive drum pad that maps keyboard inputs to drum sounds with real-time visual feedback.

## Features
- Keyboard-controlled drum sounds
- Visual feedback on key press
- Responsive layout using Tailwind CSS
- Custom background and sound assets

## Tech Stack
- HTML
- JavaScript
- Tailwind CSS

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Press keys A, S, D, F, G, H, J, K, I to play sounds
